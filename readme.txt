== Description ==

Corporate Testimonials powered by ASSIST gives your WordPress personal blog or a corporate website power to have an elegant filterable testimonials and team where you can showoff your amazing work or images. 

* You can check a live demo of this portfolio on <a href="http://corporate.assist.ro" target="_blank">our demo page</a>.
* Visit our website <a href="https://assist-software.net/" target="_blank"> https://assist-software.net</a> for more products or software development services.		

== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory, or download and install automatically through your admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
